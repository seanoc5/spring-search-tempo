Getting Started

Discovery sessions are created when a remote machine runs the onboard command.

On the remote machine:
# Download and run the remote crawler
java -jar remote-crawler.jar onboard --server http://this-server:8082

# Or specify paths directly
java -jar remote-crawler.jar onboard -d "C:,D:" --yes
Classification Workflow:
  1. Remote machine discovers folder structure
  2. Folders are uploaded here with suggested classifications
  3. You review and adjust classifications in the UI
  4. Apply classifications to create/update crawl configurations
  5. Run regular crawls with the new configuration
