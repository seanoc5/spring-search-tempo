About Hybrid Search

Hybrid search combines keyword matching (FTS) with semantic similarity for best results.

How it works
  • FTS: Finds exact keyword matches using PostgreSQL full-text search
  • Semantic: Finds conceptually similar content using vector embeddings
  • RRF: Reciprocal Rank Fusion combines rankings from both methods
Weight adjustments
  • Equal weights (0.5/0.5): Balanced search for general queries
  • Higher FTS (0.8/0.2): When you need exact keyword matches
  • Higher Semantic (0.2/0.8): When you want conceptually similar content
Result indicators
  • Both - Found by both FTS and semantic search (highest confidence)
  • FTS - Found by keyword search only
  • Semantic - Found by semantic search only
