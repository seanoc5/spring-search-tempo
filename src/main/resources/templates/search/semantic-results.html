About Semantic Search

Semantic search finds content based on meaning, not just keywords.

How it works
  • Your query is converted to a vector embedding using an AI model
  • Content chunks with similar embeddings are returned
  • Results are ranked by cosine similarity (higher = more similar)
When to use
  • Conceptual queries: "documents about machine learning optimization"
  • Paraphrased content: Find content that says the same thing differently
  • Related topics: Discover content on similar subjects
Requirements
  • Ollama must be running with the embedding model loaded
  • Content must have embeddings generated (run embedding job)
