Crawl Review Tool

Compare filesystem state against database state for a specific crawl configuration. Identify what's been crawled, what's missing, and why.

Folder Comparison

Compare folders from a filesystem list against what's indexed in the database. Shows which folders exist in both, which are missing from DB (expected or unexpected), and which are in DB but no longer on filesystem.

Upload a text file with one folder path per line.
How to generate a folder list 
# List all folders in a directory
find /home/user/Documents -type d > folders.txt

# Exclude hidden folders
find /home/user/Documents -type d -not -path '*/.*' > folders.txt
File Comparison

Compare files from a filesystem list against what's indexed. Shows coverage by file extension - how many files of each type are indexed compared to what exists on the filesystem.

Upload a text file with one file path per line.
How to generate a file list 
# List all files in a directory
find /home/user/Documents -type f > files.txt

# Only common document types
find /home/user/Documents -type f \
  \( -name "*.pdf" -o -name "*.doc*" -o -name "*.txt" \) > files.txt
