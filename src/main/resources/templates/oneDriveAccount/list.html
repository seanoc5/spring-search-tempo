Setup

To connect OneDrive:

  1. Register an app at Azure Portal (App registrations)
  2. Set platform to "Mobile and desktop applications" with redirect URI
  3. Add permissions: Files.Read, User.Read, offline_access
  4. Set ONEDRIVE_CLIENT_ID and ONEDRIVE_TOKEN_ENCRYPTION_KEY env vars
  5. Set app.onedrive.enabled: true in config
